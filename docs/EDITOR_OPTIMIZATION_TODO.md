# 笔记编辑器优化任务清单 (Editor Optimization Progress)

## 已完成任务 (Completed)

### 1. Markdown 编辑器增强
- [x] 工具栏全面优化：集成 20+ 功能按钮（标题、粗体、表格、代码块等）
- [x] 数学公式支持：集成 KaTeX，支持行内 ($...$) 和块级 ($$ ... $$) 公式渲染
- [x] 流程图/图表支持：集成 Mermaid，支持流程图、序列图、甘特图等
- [x] 实时预览优化：配置 `remark-math` 和 `rehype-katex`
- [x] 增加 Markdown 语法帮助入口

### 2. 思维导图编辑器增强
- [x] 顶部功能菜单实现：
    - [x] 节点操作：添加子节点、同级节点、删除节点
    - [x] 视图縮放：放大、缩小、适应窗口
    - [x] 样式切换：支持 8 种以上主题切换 (Classic, Dark, Blue, Fresh 等)
    - [x] 布局切换：支持 5 种布局 (逻辑结构、思维导图、鱼骨图等)
- [x] 导出功能：支持 PNG 图片导出和 JSON 数据导出
- [x] 画布操作优化：支持快捷键提示和“清空画布”功能
- [x] 专注模式优化：去除冗余边距，保持画布最大化

### 3. 富文本编辑器增强
- [x] 专业级工具栏：增加字体、字号、颜色、背景色、对齐方式、缩进等
- [x] 公式支持：集成 KaTeX 插件，支持所见即所得的公式录入
- [x] 多媒体支持：支持超链接、图片和视频嵌入

### 4. 界面与交互优化
- [x] 统一深色模式适配：所有编辑器均支持自动切换深色主题
- [x] 保存状态提示：实时显示“正在保存/已保存”状态
